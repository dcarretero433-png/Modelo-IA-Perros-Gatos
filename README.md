# Modelo-IA-Perros-Gatos
Construcción de una aplicación real de Machine Learning en Python para resolver un problema de clasificación de imágenes "Perro/Gato", usando un modelo entrenado en Teachable Machine y y ejecutado en Google Colab. El programa permite analizar imágenes individuales o recorrer un lote, comparando las predicciones del modelo con las etiquetas esperadas.
